# Cz-Admin-Message-Server

provide cz-admin's wechat services;

Stack
-----

- koa
- mongoose
- axios
- bluebird
- redis
- websocket
- .....


Quick Start
-----------

```shell
$ git clone https://github.com/jomalonejia/Cz-Admin-Message-Server.git
$ cd  Cz-Admin-Message-Server
$ npm install
$ npm start
```

server would be on localhost:3000
